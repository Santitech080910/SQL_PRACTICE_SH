# SQL Practice 📊

Repositorio personal dedicado a profundizar en SQL con enfoque en 
**análisis de datos financieros**. Aquí documento mi progreso, 
soluciones a problemas complejos y proyectos aplicados al dominio 
de finanzas.

## 🎯 Objetivo

Construir un portafolio técnico sólido que refleje dominio de SQL 
a nivel senior, combinando rigor técnico con conocimiento del 
negocio financiero.

## 📂 Estructura del repositorio

- **`01-fundamentals/`** — Joins, agregaciones, subqueries y 
  window functions.
- **`02-advanced/`** — CTEs recursivas, optimización de queries, 
  estrategias de indexación y tuning.
- **`03-finance-use-cases/`** — Cálculo de retornos, análisis de 
  riesgo, cohortes de clientes, flujos de caja y detección de 
  patrones anómalos.
- **`04-data-modeling/`** — Esquemas estrella, SCD (Slowly Changing 
  Dimensions) y diseño de data warehouses financieros.
- **`05-interview-prep/`** — Soluciones a problemas de LeetCode, 
  StrataScratch y casos de estudio.
- **`06-projects/`** — Proyectos end-to-end con datasets reales 
  o simulados.

## 🛠️ Stack técnico

- SQL (PostgreSQL / otros motores según el caso)
- Herramientas de visualización y análisis complementarias
- Datasets públicos y generados sintéticamente
